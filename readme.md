# jumangos

chaos video player system written in processing-java

# how to compile

1. install emacs
2. install package called 'processing-mode'
3. open window.pde and C-c C-r

# usage

- global

  - tab
    switch debugger on/off

  - -(MINUS)
    garbage collection
	
  - ` or ]
    overlay on and off

- play mode

  - 1-9
    play speed
  - 0
    super high speed
  - LEFT RIGHT
    switch view
  - ยง
    default view
  - UP DOWN
    switch deco
  - [
    remove deco

- debugger mode
  - 1-9
    index specifier
  - 0
    random index
  - back space
    remove fuzzy input
